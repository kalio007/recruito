## Getting Started wuth the Project

First, run the development server:

## Using docker: Run

```bash

```

## Running without docker: Run

```bash
npm run dev
```

next Run,

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
